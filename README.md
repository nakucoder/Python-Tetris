# Python Tetris Clone

A classic Tetris game built from scratch using 
Python and Pygame.

## Tech Stack
- **Language:** Python
- **Library:** Pygame

## Features
- Classic Tetris gameplay mechanics
- Collision detection system
- Grid-based state management
- Line-clearing algorithms
- Real-time user input handling
- Piece rotation physics
- Score tracking system
- Increasing difficulty levels

## Getting Started

### Prerequisites
- Python 3.x
- Pygame

### Installation
1. Clone the repository
2. Install dependencies:
   `pip install pygame`
3. Run the game:
   `python tetris.py`

## How to Play
- **Left/Right arrows** — move piece
- **Up arrow** — rotate piece
- **Down arrow** — speed up piece
- **Space** — drop piece instantly
